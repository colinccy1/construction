建筑实务备考

  • + - 招标投标管理
    • + - 一、出售招标文件--提交招标文件截止日：
      • 1.出售招标文件的时间不得少于5日
      • 2.投标人对招标文件有异议，应在提交投标文件截止日10天前提出。
      • 3.招标文件的修正：在投标截止时间15日前修改（otherwise 应顺延提交投标文件截止时间），并以书面形式通知所有招标文件收受人。
      • 4.出售招标文件至提交投标文件截止日不少于20日，5+15.
      • 5.招标控制价的规定：招标控制价应在招标文件中公布，不得保密。
      • 6.任何单位和个人不得违法限制或排斥本地区、本系统以外的法人或其他组织参加投标，不得以任何方式非法干涉投标活动。
      • 7.招标人不得组织单个或部分潜在投标人踏勘现场。
      • 8.在提交投标文件截止时间前，投标人可以修改、补充、撤回投标文件，并书面通知招标人。
    • + - 二、开标，即提交投标文件截止时间的同一时间：
      • 1.在提交投标文件截止时间后，投标人不得修改、补充投标文件；撤回投标文件的，投标保证金将会没收。
      • + - 2.标底的规定：
        • 1）标底必须保密，在开标时公布。
        • 2）标定仅作为评标参考，不得以投标报价超过标底上下浮动范围作为否决投标的条件。
      • 3.主持人就是招标人。
      • + - 4.招标人拒收(区别于否决情形）投标文件的情形：
        • 1）逾期送达的。
        • 2）未按招标文件要求密封的。
    • + - 三、评标:
      • 1.评标标准和评标方法必须在招标文件中公开载明，并不得随意更改，否则不得作为评标依据。
      • + - 2.评标委员会的组成：
        • 1）5人及以上单数。
        • 2）其中经济、技术方面专家不得少于成员总数的2/3，业主不多于1/3.
        • 3）在中标结果确认前评标委员会应保密。
      • 3.评标委员会认为需要投标人作出必要澄清、说明的，应当书面通知该投标人。评标委员会不得接受投标人主动提出的澄清、说明。
      • + - 4.评标委员会会否决投标人的投标的情形：
        • 1）无单位盖章和单位负责人签字的。
        • 2）投标价格低于成本或者高于招标文件设定的最高投标限价。
        • 3）投标文件未对招标文件的实质性要求和条件作出响应。
    • + - 四、确定中标人--签订合同：
      • + - 1.确定中标人的注意事项
        • 1）经招标人授权，评标委员会可直接确定中标人。
        • 2）在向中标人发出中标通知书的同事，将中标结果通知所有未中标的投标人。
      • 2.中标通知书发出之日起30日内，招标人应与中标人签订合同。
      • 3.在签订合同后的5日内，向招标人和未中标的投标人退还投标保证金和银行同期同类存款利息。
  • + - 项目管理相关规定
    • + - 一、项目管理规划的作用：
      • 1.制定施工项目管理的目标。
      • 2.规划实施项目目标的组织、程序和方法，落实责任。
      • 3.作为相应项目的管理规范，在项目管理过程中贯彻执行。
      • 4.作为考核项目经理部的依据之一。
    • + - 二、项目管理实施规划的编制依据
      • 1.项目管理规划大纲。
      • 2.项目条件和环境分析资料。
      • 3.工程合同及相关文件。
      • 4.同类项目的相关资料。
    • + - 三、项目职业健康安全管理：
      • 1.项目职业健康安全技术措施计划是由项目经理主持编制。
      • 2.项目职业健康安全技术措施计划的编制步骤：
        • 1）工作分类。
        • 2）识别危险源。
        • 3）确定风险。
        • 4）评价风险。
        • 5）制定风险对策。
        • 6）评审风险对策的充分性。
      • 3.项目职业健康安全技术措施计划的内容：
        • 1）工程概况。
        • 2）控制目标。
        • 3）控制程序。
        • 4）组织机构。
        • 5）职责权限。
        • 6）规章制度。
        • 7）资源配置。
        • 8）安全措施等
      • 4.项目职业健康安全技术交底：施工前，应由项目技术负责人对相关人员进行交底。
    • + - 四、项目成本管理
      • + - 1.项目部进行成本管理的内容：
        • 1）成本计划
        • 2）成本控制
        • 3）成本核算
        • 4）成本分析
        • 5）成本考核
      • + - 2.项目部成本核算的“三同步”原则：
        • 1）应坚持形象进度
        • 2）产值统计
        • 3）成本归集
    • + - 五、项目的信息管理程序：
      • 1.确定项目信息管理目标。
      • 2.进行项目信息管理策划。
      • 3.项目信息收集。
      • 4.项目信息处理。
      • 5.项目信息运用。
      • 6.项目信息管理评价。
    • + - 六、风险管理：
      • 1.项目风险管理内容：
        • 1）风险识别。
        • 2）风险评估。
        • 3）风险响应。
        • 4）风险控制。
      • 2.风险管理计划内容：
        • 1）风险管理目标。
        • 2）风险管理范围。
        • 3）可使用的风险管理办法，工具以及数据来源。
        • 4）风险分类和风险排序要求。
        • 5）风险管理的职责与权限。
        • 6）风险跟踪的要求。
        • 7）相应的资源预算。
    • + - 七、项目考核评价的定量指标：
      • 工期
      • 质量
      • 成本
      • 职业安全健康
      • 环境保护
  • 施工组织设计
    • + - 总体要求
      • 一、谁主持编制：由项目负责人主持编制。
      • 二、交底
        • 1.由项目技术负责人进行书面交底。
        • 2.向现场所有单位，包括建设单位、监理单位、（总包）项目经理部相关部门、分包单位相关负责人等。
      • 三、补充、修改施工组织设计的情形
        • 1.一般修改完善的：（签字）
          • 1）施工单位技术负责人。
          • 2）项目总监理工程师。
          • 3）建设单位项目负责人
        • 2.若专项方案经论证后，需要做重大修改的,还需要重新组织专家进行论证。
        • + - 3.施工组织设计需修改的情形：
          • 1）工程设计有重大修改。
          • 2）有关法律、法规、规范和标准实施、修订和废止。
          • 3）主要施工方法有重大调整。
          • 4）重要施工资源配置有重大调整。
          • 5）施工环境有重大变化。
    • + - 分类
      • + - 施工组织总设计（群体、单位工程）
        • 1.由项目负责人来审核。
        • 2.由施工单位技术负责人审批。
      • 安全专项施工方案
        • 危险性较大的分部分项工程
          • 达到一定规模的编制专项施工方案
            • + - 一、种类条件
              • 1.3m及以上的土方工程基坑支护降水工程。
              • 2.大模、飞模、滑膜、爬模。
              • 3.高度（层高）≥5m；跨度（柱间距）≥10m的模板支撑体系。
              • 4.高度≥24m落地式钢管脚手架；悬挑脚手架；
              • 5.建筑幕墙工程，人工挖孔桩工程。
          • 超过一定规模的编制专项施工方案并组织专家论证
            • + - 一、种类条件
              • 1.5m及以上的土方工程基坑支护降水工程。
              • 2.飞模、滑膜、爬模。
              • 3.高度（层高）≥8m；跨度（柱间距）≥18m的模板支撑体系。
              • 4.高度≥50m落地式钢管脚手架；悬挑脚手架≥20m。
              • 5.建筑幕墙工程高度≥50m，人工挖孔桩工程深度≥16m。
            • 二、专家论证的相关规定：
              • 1.实行施工总承包的,施工总承包单位组织召开专家论证会。
              • + - 2.专家组的组成：
                • 1）专家组成员应当由5名以上符合相关专业要求的专家组成，本项目参建各方人员（建设方、监理、设计等）不得以专家身份参加专家论证会。
                • + - 2）施工单位参建人员：
                  • （1）项目部：项目负责人、项目技术负责人、专项施工方案编制人员、项目专职安全生产管理人员。
                  • （2）单位总部：施工单位分管安全的负责人、技术负责人。
              • + - 3.论证内容：（了解）
                • 1）专项方案内容是否完整、可行。
                • 2）专项方案技术书和验算依据是否符合标准、规范。
                • 3）安全施工的基本条件是否满足现场实际情况。
        • + - 专项施工方案内容
          • 工程概况
          • + - 编制依据
            • 1.相关法律、法规。
            • 2.规范性文件、标准。
            • 3.规范及图集。
            • 4.施工组织设计等。
          • 施工计划
          • 施工工艺技术
          • + - 施工安全保证措施
            • 1.组织保障。
            • 2.技术措施。
            • 3.应急预案。
            • 4.监测监控。
          • 劳动力计划
          • 计算书及相关图纸
        • + - 专项施工方案编制、审核、审批：
          • + - 一、编制单位：
            • 1.实现施工总承包的，由施工总承包单位编制。
            • + - 2.由专业承包单位编制的有：
              • 1）起重机械安装拆除工程。
              • 2）深基坑工程。
              • 3）附着式升降脚手架工程。
          • 二、审核：由施工单位技术部门组织本单位技术、质量、安全等专业技术人员审核。
          • + - 三、审批：
            • + - 1.施工单位内部审批：
              • 1）一般由施工单位技术负责人审批。
              • 2）若由专业承包单位编制，那么要有专业分包单位技术负责人和施工总承包单位技术负责人签字。
            • 2.再（由总包）交项目总监理工程师审核签字。
          • 四、实施时由专职安全生产管理人员在现场监督。
        • 一般分部分项工程编制普通施工方案
  • 质量管理
    • + - 事前
      • 一、质量计划
        • 1.编制依据（从项目、企业、国家想）
          • 1）工程承包合同、设计图纸及相关文件。
          • 2）施工组织设计、专项施工方案。
          • 3）企业的质量管理体系文件。
          • 4）国家和地方相关的法律、法规、技术标准、规范及有关施工操作规程。
        • 2.审批流程：由项目经理组织编写，报企业相关管理部门批准并得到发包方和监理方认可后实施。
        • 3.质量计划内容
          • 1)质量目标和要求；
          • 2）质量管理组织和职责；
          • 3）所需的过程、文件和资源；
          • 4）记录要求；
          • 5）所采取的措施；
      • + - 二、材料
        • + - 材料质量控制的4个环节
          • 1.材料的采购。
          • 2.材料的进场实验检验。
          • 3.过程保管。
          • 4.材料使用。
        • + - 质量验证
          • + - 1.验证内容
            • 1）材料品种。
            • 2）材料型号。
            • 3）材料规格。
            • 4）材料数量。
            • 5）材料外观检查及见证取样。
          • + - 2.见证取样
            • （1）在建设单位或监理工程师的见证下。
            • （2）由项目实验员从进场材料中随机抽取送检。
            • （3）并由施工单位填写委托单。
          • 3.报监理工程师审批备案后方可使用。
        • 有关材料的检查内容
          • + - 1.钢筋进场检验内容：
            • 1）屈服强度。
            • 2）抗拉强度。
            • 3）伸长率。
            • 4）冷弯。
            • 5）单位长度重量偏差。
          • + - 2.预拌混凝土检测内容：
            • 1）预拌混凝土合格证书。
            • 2）水泥、砂子、石子、外加剂等原材复试报告和合格证。
            • 3）混凝土配合比单。
            • 4）混凝土试块强度报告。
          • + - 3.节能工程保温材料进场验收内容：
            • + - 1）墙体节能工程保温材料进场验收内容：
              • 导热系数。
              • 密度。
              • 抗压强度。
              • 压缩强度。
              • 燃烧性能。
            • + - 2）幕墙玻璃、建筑外窗进场复验内容：
              • 气密性
              • 传热系数
              • 玻璃遮阳系数
              • 可见光透射比
              • 中空玻璃露点
        • + - 因材料影响工程质量的责任的承担者。
          • 1.包工包料（材料由施工单位采购）：对于项目采购的物资，业主验证不能替代项目对所采购物资的质量责任。
          • 2.甲供：业主采购的物质，项目的验证不能取代业主对其采购物质的质量责任。
    • 事中
      • + - 一、基础工程
        • + - 土方开挖
          • 开挖原则：开槽支撑，先撑后挖，分层开挖，严禁超挖。
          • + - 深基坑开挖方法
            • 1.为使支护结构均匀受力，减少变形。
            • 2.分段、对称、分层、均衡的方式挖土。
          • + - 土方开挖过程中的检查内容
            • 平面位置
            • 水平标高
            • 边坡坡度
            • 压实度
            • 排水和降低地下水位系统
            • 随时观测周围环境
          • + - 开挖监控
            • + - 方案内容
              • 监控目的
              • 监控项目
              • 监控报警值
              • 监测方法及精度要求
              • 监测点布置
              • 监测周期
              • 工序管理和记录制度
              • 信息反馈系统
            • + - 重点对象
              • 支护结构水平位移
              • 周围建筑物
              • 地下管线变形
              • 地下水位
          • + - 安全距离
            • 堆置弃土、移动机械：土质良好时距离坑边1m以外，高度不超过1.5m。
            • + - 两种挖土机械
              • 挖土机间距应大于10m。
              • 推土机前后距离应大于8m。
            • 人工间距应大于2.5m
          • + - 开挖深度
            • 1.使用铲挖机、推土机时保留15~20cm。
            • 2.采用挖掘机时保留20~30cm。
          • + - 应急处理
            • + - 基坑渗漏水（先排后堵）
              • 坑底设沟排水。
              • 引流修补
              • 密实混凝土封堵
              • 压密注浆
              • 高压喷射注浆
            • + - 支护结构（往坑内）位移
              • 采用支护结构背后卸载
              • 加快垫层施工
              • 增加垫层厚度和加设支持
            • + - 支护结构墙背土体沉陷（往坑外位移）
              • 增设坑外回灌井
              • 坑底加固
              • 垫层随挖随浇
              • 加厚垫层
              • 采用配筋垫层
              • 设置坑底支撑
            • + - 周围管线保护
              • 增设回灌井
              • 搭设封闭桩
              • 开挖隔离沟
              • 管线架空
          • + - 地基处理
            • + - 1.常用的地基处理方法：
              • 换填垫层法
              • 预压法
              • 石灰桩法
              • 振冲法
              • 砂石桩法
            • + - 2.选择最佳地基处理方法的依据：
              • 加固原理
              • 适用范围
              • 耗用材料
              • 施工机械
              • 工期要求
              • 对环境的影响
            • + - 3.强夯处理和强夯置换法
              • 1）强夯处理范围
                • 强夯处理范围应大于建筑物基础范围。
                • 每边超出基础外缘宽度宜为基底下设计设计处理深度的1/2~2/3,且不宜小于3m。
              • 2）检查内容
                • 夯实前：夯捶质量、尺寸、落距控制手段、排水措施、被夯地基土质。
                • 夯实施工过程中：落距、夯击遍数、夯点位置、夯击范围。
          • + - 质量问题
            • 回填土密实度达不到要求的原因及治理
              • + - 原因
                • 1.土的含水率过大或过小，因而达不到最优含水率下的密实度要求。
                • 2.填方土料不符合要求。
                • 3.碾压或夯实机具能量不够，达不到影响深度要求，使土的密实度降低。
              • + - 治理
                • 1.将不符合要求的土料挖出换土，或掺入石灰、碎石等夯实加固。
                • 2.因含水量过大达不到密实度的土层，可采用翻送晾晒、风干，或均匀掺入干土等吸水材料，重新夯实。
                • 3.因含水量小或碾压机能量过小时，可采用增加夯实遍数，或使用大功率压实机碾压等措施。
            • + - 基坑槽泡水
              • 1.采取措施，将水引走排净。
              • 2.设置截水沟，防止水刷边坡。
              • 3.已被水浸泡扰动的土，采取排水晾晒后夯实，或抛填碎石、小块石夯实，或换土夯实（3：7灰土）。
        • + - 护壁支撑
          • 1.在拆除护壁支撑时，应按照回填顺序，从下而上、逐步拆除；
          • 2.更换护壁支撑时，必须先安装新的、再拆除旧的。
        • + - 降水
          • 1.降水井形状及间距:宜在基坑外缘采用封闭式布置，在地下水位补给方向应适当加密。
          • 2.降水深度：应使地下水稳定在坑底0.5m以下。
          • 3.持续时间：降水工作应持续到基础（包括地下水位下回填土）施工完成。
        • + - 泥浆护壁灌注桩
          • 浇筑混凝土前孔底沉渣厚度测量，是二次清孔后的结果。
          • + - 桩的偏差、桩径、桩位、桩长
            • 1.桩径偏差值±50mm。
            • + - 2.桩径≤1m时，桩位偏差值
              • 1）边桩：D/6且不大于100mm
              • 2）中间桩：D/4且不大于150mm。
            • 3.桩长的偏差值：水下灌注混凝土时，桩顶混凝土标高至少要比设计标高高0.8~1m。
        • + - 验槽
          • 钎探法
            • + - 1.注意要点
              • 1）钎杆每打入30cm记录一次锤击数。钎探深度以设计为依据。
              • 2）设计无规定时，一般钎点按纵横间距1.5m梅花形布置，深度为2.1m。
            • + - 2.纤眼的处理步骤
              • 1）钎孔经相关人员检查、记录后；
              • 2）用盖孔块盖住孔眼；
              • 3）当设计、勘察、施工方共同验槽办理完验收手续后；
              • 4）方可用砂灌孔。
          • + - 验槽重点部位及发现异常处理原则
            • 1.重点：柱基、墙角、承重墙下和其他受力较大部位。
            • 2.如有异常，施工单位会同勘察、设计等有关单位进行处理。
        • + - 大体积混凝土工程
          • 养护方法
            • + - 保湿自然养护
              • 养护时机:混凝土终凝前，浇筑完毕后8~12h自然养护。
              • 养护时间：不少于14d。
            • + - 保温养护、温控指标
              • 1.入模温度要求：
                • 1）不宜大于30℃。
                • 2）最大升温值不宜大于50℃。
              • 2.表面以内4~10cm位置温度要求：
                • 1）覆盖或带模养护阶段（即拆模前），该部位与浇注体表面温度差值不宜大于25℃。
                • 2）结束覆盖或拆模后，该部位与环境温度差值不宜大于25℃。
              • 3.浇筑体内部相邻两测温点的温度差值不应大于25℃。
              • 4.混凝土的降温速率不宜大于2℃/d。
          • 后浇带
            • 尺寸、钢筋贯通
              • 1.筏型基础和箱型基础长度超过40m时，宜设置贯通后浇带，且宽度不宜小于80cm。
              • 2.钢筋必须贯通，防水要求，微膨胀混凝土，强度高一级。
            • 养护时间：不得少于28d。
        • + - 土方回填
          • 回填方法（分层、同时）：在基坑相对两侧或周围同时进行分层回填和夯实。
          • 回填时检查内容
            • 1.排水措施。
            • 2.每层填筑厚度。
            • 3.回填土的含水量控制。
            • 4.压实程度。
