建筑实务备考

  • + - 招标投标管理
    • + - 一、出售招标文件--提交招标文件截止日：
      • 1.出售招标文件的时间不得少于5日
      • 2.投标人对招标文件有异议，应在提交投标文件截止日10天前提出。
      • 3.招标文件的修正：在投标截止时间15日前修改（otherwise 应顺延提交投标文件截止时间），并以书面形式通知所有招标文件收受人。
      • 4.出售招标文件至提交投标文件截止日不少于20日，5+15.
      • 5.招标控制价的规定：招标控制价应在招标文件中公布，不得保密。
      • 6.任何单位和个人不得违法限制或排斥本地区、本系统以外的法人或其他组织参加投标，不得以任何方式非法干涉投标活动。
      • 7.招标人不得组织单个或部分潜在投标人踏勘现场。
      • 8.在提交投标文件截止时间前，投标人可以修改、补充、撤回投标文件，并书面通知招标人。
    • + - 二、开标，即提交投标文件截止时间的同一时间：
      • 1.在提交投标文件截止时间后，投标人不得修改、补充投标文件；撤回投标文件的，投标保证金将会没收。
      • + - 2.标底的规定：
        • 1）标底必须保密，在开标时公布。
        • 2）标定仅作为评标参考，不得以投标报价超过标底上下浮动范围作为否决投标的条件。
      • 3.主持人就是招标人。
      • + - 4.招标人拒收(区别于否决情形）投标文件的情形：
        • 1）逾期送达的。
        • 2）未按招标文件要求密封的。
    • + - 三、评标:
      • 1.评标标准和评标方法必须在招标文件中公开载明，并不得随意更改，否则不得作为评标依据。
      • + - 2.评标委员会的组成：
        • 1）5人及以上单数。
        • 2）其中经济、技术方面专家不得少于成员总数的2/3，业主不多于1/3.
        • 3）在中标结果确认前评标委员会应保密。
      • 3.评标委员会认为需要投标人作出必要澄清、说明的，应当书面通知该投标人。评标委员会不得接受投标人主动提出的澄清、说明。
      • + - 4.评标委员会会否决投标人的投标的情形：
        • 1）无单位盖章和单位负责人签字的。
        • 2）投标价格低于成本或者高于招标文件设定的最高投标限价。
        • 3）投标文件未对招标文件的实质性要求和条件作出响应。
    • + - 四、确定中标人--签订合同：
      • + - 1.确定中标人的注意事项
        • 1）经招标人授权，评标委员会可直接确定中标人。
        • 2）在向中标人发出中标通知书的同事，将中标结果通知所有未中标的投标人。
      • 2.中标通知书发出之日起30日内，招标人应与中标人签订合同。
      • 3.在签订合同后的5日内，向招标人和未中标的投标人退还投标保证金和银行同期同类存款利息。
  • + - 项目管理相关规定
    • + - 一、项目管理规划的作用：
      • 1.制定施工项目管理的目标。
      • 2.规划实施项目目标的组织、程序和方法，落实责任。
      • 3.作为相应项目的管理规范，在项目管理过程中贯彻执行。
      • 4.作为考核项目经理部的依据之一。
    • + - 二、项目管理实施规划的编制依据
      • 1.项目管理规划大纲。
      • 2.项目条件和环境分析资料。
      • 3.工程合同及相关文件。
      • 4.同类项目的相关资料。
    • + - 三、项目职业健康安全管理：
      • 1.项目职业健康安全技术措施计划是由项目经理主持编制。
      • 2.项目职业健康安全技术措施计划的编制步骤：
        • 1）工作分类。
        • 2）识别危险源。
        • 3）确定风险。
        • 4）评价风险。
        • 5）制定风险对策。
        • 6）评审风险对策的充分性。
      • 3.项目职业健康安全技术措施计划的内容：
        • 1）工程概况。
        • 2）控制目标。
        • 3）控制程序。
        • 4）组织机构。
        • 5）职责权限。
        • 6）规章制度。
        • 7）资源配置。
        • 8）安全措施等
      • 4.项目职业健康安全技术交底：施工前，应由项目技术负责人对相关人员进行交底。
    • + - 四、项目成本管理
      • + - 1.项目部进行成本管理的内容：
        • 1）成本计划
        • 2）成本控制
        • 3）成本核算
        • 4）成本分析
        • 5）成本考核
      • + - 2.项目部成本核算的“三同步”原则：
        • 1）应坚持形象进度
        • 2）产值统计
        • 3）成本归集
    • + - 五、项目的信息管理程序：
      • 1.确定项目信息管理目标。
      • 2.进行项目信息管理策划。
      • 3.项目信息收集。
      • 4.项目信息处理。
      • 5.项目信息运用。
      • 6.项目信息管理评价。
    • + - 六、风险管理：
      • 1.项目风险管理内容：
        • 1）风险识别。
        • 2）风险评估。
        • 3）风险响应。
        • 4）风险控制。
      • 2.风险管理计划内容：
        • 1）风险管理目标。
        • 2）风险管理范围。
        • 3）可使用的风险管理办法，工具以及数据来源。
        • 4）风险分类和风险排序要求。
        • 5）风险管理的职责与权限。
        • 6）风险跟踪的要求。
        • 7）相应的资源预算。
    • + - 七、项目考核评价的定量指标：
      • 工期
      • 质量
      • 成本
      • 职业安全健康
      • 环境保护
  • 施工组织设计
    • + - 总体要求
      • 一、谁主持编制：由项目负责人主持编制。
      • 二、交底
        • 1.由项目技术负责人进行书面交底。
        • 2.向现场所有单位，包括建设单位、监理单位、（总包）项目经理部相关部门、分包单位相关负责人等。
      • 三、补充、修改施工组织设计的情形
        • 1.一般修改完善的：（签字）
          • 1）施工单位技术负责人。
          • 2）项目总监理工程师。
          • 3）建设单位项目负责人
        • 2.若专项方案经论证后，需要做重大修改的,还需要重新组织专家进行论证。
        • + - 3.施工组织设计需修改的情形：
          • 1）工程设计有重大修改。
          • 2）有关法律、法规、规范和标准实施、修订和废止。
          • 3）主要施工方法有重大调整。
          • 4）重要施工资源配置有重大调整。
          • 5）施工环境有重大变化。
    • + - 分类
      • + - 施工组织总设计（群体、单位工程）
        • 1.由项目负责人来审核。
        • 2.由施工单位技术负责人审批。
      • 安全专项施工方案
        • 危险性较大的分部分项工程
          • 达到一定规模的编制专项施工方案
            • + - 一、种类条件
              • 1.3m及以上的土方工程基坑支护降水工程。
              • 2.大模、飞模、滑膜、爬模。
              • 3.高度（层高）≥5m；跨度（柱间距）≥10m的模板支撑体系。
              • 4.高度≥24m落地式钢管脚手架；悬挑脚手架；
              • 5.建筑幕墙工程，人工挖孔桩工程。
          • 超过一定规模的编制专项施工方案并组织专家论证
            • + - 一、种类条件
              • 1.5m及以上的土方工程基坑支护降水工程。
              • 2.飞模、滑膜、爬模。
              • 3.高度（层高）≥8m；跨度（柱间距）≥18m的模板支撑体系。
              • 4.高度≥50m落地式钢管脚手架；悬挑脚手架≥20m。
              • 5.建筑幕墙工程高度≥50m，人工挖孔桩工程深度≥16m。
            • 二、专家论证的相关规定：
              • 1.实行施工总承包的,施工总承包单位组织召开专家论证会。
              • + - 2.专家组的组成：
                • 1）专家组成员应当由5名以上符合相关专业要求的专家组成，本项目参建各方人员（建设方、监理、设计等）不得以专家身份参加专家论证会。
                • + - 2）施工单位参建人员：
                  • （1）项目部：项目负责人、项目技术负责人、专项施工方案编制人员、项目专职安全生产管理人员。
                  • （2）单位总部：施工单位分管安全的负责人、技术负责人。
              • + - 3.论证内容：（了解）
                • 1）专项方案内容是否完整、可行。
                • 2）专项方案技术书和验算依据是否符合标准、规范。
                • 3）安全施工的基本条件是否满足现场实际情况。
        • + - 专项施工方案内容
          • 工程概况
          • + - 编制依据
            • 1.相关法律、法规。
            • 2.规范性文件、标准。
            • 3.规范及图集。
            • 4.施工组织设计等。
          • 施工计划
          • 施工工艺技术
          • + - 施工安全保证措施
            • 1.组织保障。
            • 2.技术措施。
            • 3.应急预案。
            • 4.监测监控。
          • 劳动力计划
          • 计算书及相关图纸
        • + - 专项施工方案编制、审核、审批：
          • + - 一、编制单位：
            • 1.实现施工总承包的，由施工总承包单位编制。
            • + - 2.由专业承包单位编制的有：
              • 1）起重机械安装拆除工程。
              • 2）深基坑工程。
              • 3）附着式升降脚手架工程。
          • 二、审核：由施工单位技术部门组织本单位技术、质量、安全等专业技术人员审核。
          • + - 三、审批：
            • + - 1.施工单位内部审批：
              • 1）一般由施工单位技术负责人审批。
              • 2）若由专业承包单位编制，那么要有专业分包单位技术负责人和施工总承包单位技术负责人签字。
            • 2.再（由总包）交项目总监理工程师审核签字。
          • 四、实施时由专职安全生产管理人员在现场监督。
        • 一般分部分项工程编制普通施工方案
  • 质量管理
    • + - 事前
      • 一、质量计划
        • 1.编制依据（从项目、企业、国家想）
          • 1）工程承包合同、设计图纸及相关文件。
          • 2）施工组织设计、专项施工方案。
          • 3）企业的质量管理体系文件。
          • 4）国家和地方相关的法律、法规、技术标准、规范及有关施工操作规程。
        • 2.审批流程：由项目经理组织编写，报企业相关管理部门批准并得到发包方和监理方认可后实施。
        • 3.质量计划内容
          • 1)质量目标和要求；
          • 2）质量管理组织和职责；
          • 3）所需的过程、文件和资源；
          • 4）记录要求；
          • 5）所采取的措施；
      • + - 二、材料
        • + - 材料质量控制的4个环节
          • 1.材料的采购。
          • 2.材料的进场实验检验。
          • 3.过程保管。
          • 4.材料使用。
        • + - 质量验证
          • + - 1.验证内容
            • 1）材料品种。
            • 2）材料型号。
            • 3）材料规格。
            • 4）材料数量。
            • 5）材料外观检查及见证取样。
          • + - 2.见证取样
            • （1）在建设单位或监理工程师的见证下。
            • （2）由项目实验员从进场材料中随机抽取送检。
            • （3）并由施工单位填写委托单。
          • 3.报监理工程师审批备案后方可使用。
        • 有关材料的检查内容
          • + - 1.钢筋进场检验内容：
            • 1）屈服强度。
            • 2）抗拉强度。
            • 3）伸长率。
            • 4）冷弯。
            • 5）单位长度重量偏差。
          • + - 2.预拌混凝土检测内容：
            • 1）预拌混凝土合格证书。
            • 2）水泥、砂子、石子、外加剂等原材复试报告和合格证。
            • 3）混凝土配合比单。
            • 4）混凝土试块强度报告。
          • + - 3.节能工程保温材料进场验收内容：
            • + - 1）墙体节能工程保温材料进场验收内容：
              • 导热系数。
              • 密度。
              • 抗压强度。
              • 压缩强度。
              • 燃烧性能。
            • + - 2）幕墙玻璃、建筑外窗进场复验内容：
              • 气密性
              • 传热系数
              • 玻璃遮阳系数
              • 可见光透射比
              • 中空玻璃露点
        • + - 因材料影响工程质量的责任的承担者。
          • 1.包工包料（材料由施工单位采购）：对于项目采购的物资，业主验证不能替代项目对所采购物资的质量责任。
          • 2.甲供：业主采购的物质，项目的验证不能取代业主对其采购物质的质量责任。
    • + - 事中
      • + - 一、基础工程
        • + - 土方开挖
          • 开挖原则：开槽支撑，先撑后挖，分层开挖，严禁超挖。
          • + - 深基坑开挖方法
            • 1.为使支护结构均匀受力，减少变形。
            • 2.分段、对称、分层、均衡的方式挖土。
          • + - 土方开挖过程中的检查内容
            • 平面位置
            • 水平标高
            • 边坡坡度
            • 压实度
            • 排水和降低地下水位系统
            • 随时观测周围环境
          • + - 开挖监控
            • + - 方案内容
              • 监控目的
              • 监控项目
              • 监控报警值
              • 监测方法及精度要求
              • 监测点布置
              • 监测周期
              • 工序管理和记录制度
              • 信息反馈系统
            • + - 重点对象
              • 支护结构水平位移
              • 周围建筑物
              • 地下管线变形
              • 地下水位
          • + - 安全距离
            • 堆置弃土、移动机械：土质良好时距离坑边1m以外，高度不超过1.5m。
            • + - 两种挖土机械
              • 挖土机间距应大于10m。
              • 推土机前后距离应大于8m。
            • 人工间距应大于2.5m
          • + - 开挖深度
            • 1.使用铲挖机、推土机时保留15~20cm。
            • 2.采用挖掘机时保留20~30cm。
          • + - 应急处理
            • + - 基坑渗漏水（先排后堵）
              • 坑底设沟排水。
              • 引流修补
              • 密实混凝土封堵
              • 压密注浆
              • 高压喷射注浆
            • + - 支护结构（往坑内）位移
              • 采用支护结构背后卸载
              • 加快垫层施工
              • 增加垫层厚度和加设支持
            • + - 支护结构墙背土体沉陷（往坑外位移）
              • 增设坑外回灌井
              • 坑底加固
              • 垫层随挖随浇
              • 加厚垫层
              • 采用配筋垫层
              • 设置坑底支撑
            • + - 周围管线保护
              • 增设回灌井
              • 搭设封闭桩
              • 开挖隔离沟
              • 管线架空
          • + - 地基处理
            • + - 1.常用的地基处理方法：
              • 换填垫层法
              • 预压法
              • 石灰桩法
              • 振冲法
              • 砂石桩法
            • + - 2.选择最佳地基处理方法的依据：
              • 加固原理
              • 适用范围
              • 耗用材料
              • 施工机械
              • 工期要求
              • 对环境的影响
            • + - 3.强夯处理和强夯置换法
              • 1）强夯处理范围
                • 强夯处理范围应大于建筑物基础范围。
                • 每边超出基础外缘宽度宜为基底下设计设计处理深度的1/2~2/3,且不宜小于3m。
              • 2）检查内容
                • 夯实前：夯捶质量、尺寸、落距控制手段、排水措施、被夯地基土质。
                • 夯实施工过程中：落距、夯击遍数、夯点位置、夯击范围。
          • + - 质量问题
            • 回填土密实度达不到要求的原因及治理
              • + - 原因
                • 1.土的含水率过大或过小，因而达不到最优含水率下的密实度要求。
                • 2.填方土料不符合要求。
                • 3.碾压或夯实机具能量不够，达不到影响深度要求，使土的密实度降低。
              • + - 治理
                • 1.将不符合要求的土料挖出换土，或掺入石灰、碎石等夯实加固。
                • 2.因含水量过大达不到密实度的土层，可采用翻送晾晒、风干，或均匀掺入干土等吸水材料，重新夯实。
                • 3.因含水量小或碾压机能量过小时，可采用增加夯实遍数，或使用大功率压实机碾压等措施。
            • + - 基坑槽泡水
              • 1.采取措施，将水引走排净。
              • 2.设置截水沟，防止水刷边坡。
              • 3.已被水浸泡扰动的土，采取排水晾晒后夯实，或抛填碎石、小块石夯实，或换土夯实（3：7灰土）。
        • + - 护壁支撑
          • 1.在拆除护壁支撑时，应按照回填顺序，从下而上、逐步拆除；
          • 2.更换护壁支撑时，必须先安装新的、再拆除旧的。
        • + - 降水
          • 1.降水井形状及间距:宜在基坑外缘采用封闭式布置，在地下水位补给方向应适当加密。
          • 2.降水深度：应使地下水稳定在坑底0.5m以下。
          • 3.持续时间：降水工作应持续到基础（包括地下水位下回填土）施工完成。
        • + - 泥浆护壁灌注桩
          • 浇筑混凝土前孔底沉渣厚度测量，是二次清孔后的结果。
          • + - 桩的偏差、桩径、桩位、桩长
            • 1.桩径偏差值±50mm。
            • + - 2.桩径≤1m时，桩位偏差值
              • 1）边桩：D/6且不大于100mm
              • 2）中间桩：D/4且不大于150mm。
            • 3.桩长的偏差值：水下灌注混凝土时，桩顶混凝土标高至少要比设计标高高0.8~1m。
        • + - 验槽
          • 钎探法
            • + - 1.注意要点
              • 1）钎杆每打入30cm记录一次锤击数。钎探深度以设计为依据。
              • 2）设计无规定时，一般钎点按纵横间距1.5m梅花形布置，深度为2.1m。
            • + - 2.纤眼的处理步骤
              • 1）钎孔经相关人员检查、记录后；
              • 2）用盖孔块盖住孔眼；
              • 3）当设计、勘察、施工方共同验槽办理完验收手续后；
              • 4）方可用砂灌孔。
          • + - 验槽重点部位及发现异常处理原则
            • 1.重点：柱基、墙角、承重墙下和其他受力较大部位。
            • 2.如有异常，施工单位会同勘察、设计等有关单位进行处理。
        • + - 大体积混凝土工程
          • 养护方法
            • + - 保湿自然养护
              • 养护时机:混凝土终凝前，浇筑完毕后8~12h自然养护。
              • 养护时间：不少于14d。
            • + - 保温养护、温控指标
              • 1.入模温度要求：
                • 1）不宜大于30℃。
                • 2）最大升温值不宜大于50℃。
              • 2.表面以内4~10cm位置温度要求：
                • 1）覆盖或带模养护阶段（即拆模前），该部位与浇注体表面温度差值不宜大于25℃。
                • 2）结束覆盖或拆模后，该部位与环境温度差值不宜大于25℃。
              • 3.浇筑体内部相邻两测温点的温度差值不应大于25℃。
              • 4.混凝土的降温速率不宜大于2℃/d。
          • 后浇带
            • 尺寸、钢筋贯通
              • 1.筏型基础和箱型基础长度超过40m时，宜设置贯通后浇带，且宽度不宜小于80cm。
              • 2.钢筋必须贯通，防水要求，微膨胀混凝土，强度高一级。
            • 养护时间：不得少于28d。
        • 土方回填
          • 回填方法（分层、同时）：在基坑相对两侧或周围同时进行分层回填和夯实。
          • 回填时检查内容
            • 1.排水措施。
            • 2.每层填筑厚度。
            • 3.回填土的含水量控制。
            • 4.压实程度。
      • + - 二、主体工程
        • + - 1.混凝土结构子分部工程
          • + - 模板工程
            • 支撑系统的立柱底部应设置木垫板，禁止使用砖及脆性材料铺垫，目的是合理传递荷载。
            • + - 拆除
              • 1.拆模之前必须填写拆模申请，并在同条件养护试块强度达到规定要求时，施工单位项目技术负责人批 准拆模。
              • 2.根据模板设计要求进行;如模板设计无具体要求，可按先支的后拆，后支的先拆，先拆非承重模板，后 拆承重模板及支架。
              • 3.混凝土强度
                • + - (1) 拆除底模和支撑时混凝土强度应符合设计要求，当设计无要求时， 混凝土强度应符合下列规定:
                  • ①板: 三二2日1， 50%; 2m<X运8m， 75%; >8日1， 100%。
                  • ② ( 简支)梁: 三二8日1， 75%; >8m, 100%。
                  • ③悬臂梁: 100%。
                • (2) 拆除侧模时，混凝土强度应能保证其表面及棱角不受损伤。
              • 4. 模板及配件应放入室内和敞棚内，当必须露天堆放时，底部应垫高10伽m，顶面应遮盖防水篷布或塑 料布。
          • + - 钢筋工程
            • + - 加工
              • ( 1) 钢筋加工包括调直、除锈、下料切断、接长、弯曲成型等。
              • ( 2) 钢筋冷拉调直的伸长率:
                • ①光圆钢筋冷拉率不宣大子4%。
                • ②带肋钢筋冷拉率不宣大于1%。
              • ( 3) 钢筋除锈方法:
                • ①冷拉或调直过程中除锈;
                • ②可采用机械除锈、喷彤、除锈、酸洗除锈、手工除锈。
            • 钢筋的接头
              • + - 1.连续梁、板钢筋的接头位置:
                • ①直设置在受力较小处;
                  • 上部纵向筋接头位置直设置在跨中1/3 跨度范围内:
                  • 下部纵向受力钢筋在梁端1/3 跨度范围内。
                • ②接头末端至钢筋弯起点的距离不应小子钢筋直径10 倍。
              • 2. 接头数量:同一纵向受力钢筋不直设置两个及两个以上接头。
            • 钢筋代换
              • 代换原则
                • ①等强度:当构件配筋受强度控制时，按钢筋代换前后强度相等原则进行代换。
                • ②等面积: 赵爱林 当构件按最小配筋率配筋、或同钢号钢筋之间的代换， 按钢筋代换前后面积相等的原则进行代换。
              • 钢筋代换应征得设计单位的同意并办理相应手续(书面) 。
          • + - 混凝土工程
            • 混凝土的搅拌一一装料顺序:石子→水泥→砂子→水。
            • + - 浇筑
              • 1.浇筑前的检查内容:对模板、支撑、钢筋、预埋件等认真细致检查。
              • + - 2. 浇筑梁、板混凝土时的注意要点:
                • ①应在柱、墙浇筑完毕后{亭歇1"'1. 5h，浇筑梁、板。
                • ②主次梁的楼板，宣JI顶着次梁方向浇筑，单向板直沿着板的长边方向浇筑。
              • + - 3 . 施工缝处继续浇筑混凝土的相关规定:
                • ①己浇筑的混凝土，其抗压强度不应小于1. 2N/nun2 。
                • ②在己硬化的混凝土表面， 应清除水泥薄膜和松动的石子以及软弱混凝土层，并加以充分湿润和冲洗， 不得积水。
                • ③在浇筑混凝土前，宣先在施工缝处铺一层水泥浆(可掺适量界面剂)或铺一层与混凝土内成 分相同的水泥砂浆。
                • ④混凝土细致捣实，使新旧混凝土紧密结合、加强养护。
            • + - 养护
              • 1.养护时机:应在混凝土终凝前， 通常为混凝土浇筑完毕后矿12h 内开始进行自然养护。
              • 2. 需要养护14d 的情形: 掺用缓凝剂、有抗渗要求、掺矿物掺合料的混凝土。
              • + - 3.总结养护时间
                • 1.刊的情形:采用普通水泥、硅酸盐水泥、矿渣水泥拌制的混凝土。
                • + - 2. 14d 的情形:
                  • ①普通混凝土的3 种情形。
                  • ②大体积混凝土。
                  • ③防水混凝土和砂浆。
                  • ④一般后浇带混凝土的养护。
                • 3 . 28d 的情形:防水混凝土的后浇带混凝土的养护。
            • + - 后浇带
              • 1.留置时间: 后浇带通常根据设计要求留设，并保留一段时间，如设计无具体要求，则至少保留28d 后再浇筑混凝土， 将结构连成整体。
              • 2. 后浇带的填充要求:  用微膨胀混凝土，强度等级比原结构强度提高一级，并保持至少14d (防水混凝土的后浇带养护28d) 的湿润养护。
            • 特殊季节
              • + - 1.冬期施工(连续5 天最低气温低于5 0C) :
                • ①应优先采用加热水的方法，严禁加热水泥、掺合料。
                • ②预拌混凝土和远距离运输混凝土拌合物出机温度不宣低于150C ，混凝土入模温度不应低于50C 。
                • ③分层浇筑时，分层厚度不应小子400mlll; 且上层混凝土覆盖前，己浇筑混凝土温度不得低于2 0C 。
              • + - 2. 高温季节施工( 连续5 天最高温度超过300C):
                • 采用白色涂装的混凝土搅拌运输车运输
                • 混凝土入模温度不应高于35 0C 。
            • 质量问题处理
              • + - 1.混凝土强度等级偏低的原因。
                • ①混凝土原材料不符合国家标准的规定。
                • ②没有法定检测单位提供的混凝土配合比试验报告，或操作中没有严格按照混凝土配合比进行规范操 作。
                • ③拌制混凝土时投料计量有误。
                • ④混凝土搅拌、运输、浇筑、养护不符合规范要求。
              • + - 2.混凝土强度等级偏低的防治。
                • ①拌制混凝土所用的原材料必须符合有关标准规定。
                • ②按法定检测单位发出的混凝土配合比试验报告配制。
                • ③配制混凝土必须按质量比计量投料、且计量要准确。
                • ④采用机械搅拌，加料顺序正确，并严格控制搅拌时间。
                • ⑤混凝土的运输与浇捣必须在混凝土初凝前进行。
                • ⑥控制好混凝土的浇筑和振捣质量。
                • ⑦控制好混凝土的养护。
              • 3. 混凝土表面缺陷:
                • ( 1) 现象:出现麻面、露筋、蜂窝、孔洞。
                • + - ( 2) 产生的原因
                  • ①模板表面不光滑、安装质量差，拼缝不严、漏浆，模板表面污染物未清除。
                  • ②木模板在混凝土入模之 前没有充分湿润，钢模板隔离剂涂刷不均匀。
                  • ③钢筋保护层垫块厚度或放置间距、位置不当。
                  • ④混凝土明落度、和易性不好。
                  • ⑤混凝土浇筑方法不当、不分层或分层过厚，布料顺序不合理。
                  • ⑥混凝土浇筑高度超过规定要求，且未 采取措施，导致混凝土离析。
                  • ⑦漏振或振捣不实。
                  • ③混凝土拆模过早。
        • 2.脚手架工程
          • + - 立杆的安装要点:
            • 1.立杆底部应设置木垫板。
            • 2 . 立杆的接长:
              • ①除顶层顶步可搭接外，其余接头必须采用对接扣件连接。
              • ②立杆上的对接扣件必须交错布置，相邻两立柱的对接接头不得在同步内，在高度方向上错开的距离不 应小子500mm。
              • ③各接头中心距最近的主节点不直大子步距的1/3 。
            • + - 3. 立杆基础标高不在同一高度时:
              • ①高低差不应大于1m。
              • ②必须将高处的纵向扫地杆向低处延长两跨与立杆固定。
              • ③靠边坡上方的立杆轴线到边坡的距离不应小子500mm。
          • + - 扫地杆的设置:
            • 1.纵向扫地杆应距底座上皮不大于200mm;
            • 2 . 横向扫地杆应固定在紧靠纵向扫地杆下方的立杆上。
          • + - 纵向水平杆的接长:
            • 1.对接接头设置要点:
              • ①两根相邻纵向水平杆的接头不应设置在同跨内，两个相邻接头在水平方向上错开的距离不应小于 500mm;
              • ②各接头中心至最近主节点的距离不应大子纵距的1/3。
            • 2 . 搭接设置要点:搭接长度不应小子1m，应设置3 个旋转扣件固定。
          • + - 脚手架架体与建筑物的连接方式:
            • 1.高度在24m 以下脚手架，严禁使用仅有拉筋的柔性连墙件。
              • ①可采用刚性连墙件与建筑物可靠连接;
              • ②也可采用拉筋与顶撑配合使用的附墙连接方式。
            • 2. 高度在24m 以上的脚手架:必须采用刚性连墙件与建筑物可靠连接。
          • + - 剪刀撑的设置:
            • 1.高度在24m 以下的脚手架:必须在外侧两端、中间不超过15m 的立面上， 各设置一道剪刀撑，由底至 顶连续设置。
            • 2. 24m 以上的脚手架:应在外侧全立面连续设置剪刀撑。
          • + - 脚手架的拆除要点:
            • 1.禁止交叉作业:拆除作业必须由上而下逐层进行，严禁上下同时作业。
            • 2. 连墙件的拆除:严禁先将连墙件整层拆除后再拆除脚手架，连墙件必须随脚手架逐步拆除。
            • 3. 分段拆除注意要点:分段拆除高差不应大于2 步，否则应增设连墙件加固。
        • + - 3.砌体工程
          • 砌筑砂浆的使用要点:
            • 1.使用时间: 3h; 30.C 以上， 2h 。
            • 2. 施工中不应采用强度等级小子师的水泥砂浆替代同强度等级水泥混合砂浆，如需代替，应将水泥砂 浆提高一级。
          • + - 各种砌块的砌筑要点:
            • 1.多孔砖: 多孔砖的孔洞应垂直子受压面砌筑，半盲孔多孔砖的封底面应朝上砌筑。
            • + - 2. 烧结空心砖:
              • ( 1) 砌筑时，空心砖孔洞应呈水平方向，门窗洞口一砖范围内应采用烧结普通砖砌筑。
              • ( 2) 空心砖墙底部宣砌3 皮烧结普通砖。
              • ( 3) 留搓的规定:
                • ① ( 墙体中部)不得留搓，中途停歇时，应将墙顶砌平。
                • ②转角处及交接处应同时砌筑，不得留直搓; 留斜搓时，其高度不宣大于1. 2m。
          • 填充墙砌筑要点:填充墙梁下口最后3 皮砖应在下部墙砌完14d 后砌筑，并由中间开始向两边斜砌。
          • 脚手眼补砌时，脚手眼处砖及填塞用砖应湿润，并填实砂浆，不得用干砖填塞。
      • + - 三、防水工程
        • + - 一、地下防水
          • 地下防水工程必须由取得相应资质的专业防水队伍进行施工，主要施工人员应持有职业资格证书。
          • + - 防水混凝土施工要点:
            • 1.抗渗等级要求:
              • ①防水混凝土抗渗等级不小子P6 ;
              • ②防水混凝土试配时抗渗等级应比设计要求提高O. 2MPa:，并应根据试配的结果确定防水混凝土的最终 配合比。
            • 2. 当明落度损失后不能满足施工要求时的处理:
              • ①应加入原水胶比的水泥浆。
              • ②二次掺加减水剂进行搅拌，严禁直接加水。
          • + - 质量问题产生的原因及治理:
            • + - 1.防水混凝土施工缝渗漏水:
              • (1) 原因:
                • ①施工缝留置位置不当。
                • ②支模和绑钢筋过程中，掉入缝内的杂物没有及时清除，浇筑上层混凝土后在新旧混凝土之间形成夹层。
                • ③在浇筑上层混凝土时，未按规定处理施工缝，上下层混凝土不能车圄粘贴。
                • ④钢筋过密，混凝土浇筑困难，施工质量不易保证。
                • ⑤下料方法不当，骨料集中子施工缝处。
              • (2) 不渗漏的施工缝治理方法:可沿缝剔成八字形凹槽， 将松散石子剔除，刷洗干净，用水泥素浆打底，抹1:2.5 水泥砂浆找平压实。
            • 2. 防水混凝土裂缝渗漏水，产生裂缝的原因:
              • ①混凝土搅拌不均匀或水泥品种混用，收缩不一产生裂缝。
              • ②设计中，对土的侧压力及水压作用考虑不周，结构缺乏足够的刚度。
              • ③由于设计或施工等原因产生局部断裂或环形裂缝。
        • + - 二、室内防水
          • 填充墙砌筑要点:有防水要求的填充墙体采用小砌块、加气块砌筑时，墙底部宣现浇混凝土坎台，其高度直为150mm。
          • + - 防水涂料施工要求:
            • 涂膜防水应多遍成活，后一遍涂料施工应在前一遍涂层干燥后再进行，
            • 前后两遍涂刷方向应相互垂直， 并直先涂刷立面、后涂刷平面。
          • 室内防水的蓄水试验:
            • + - 1.地面蓄水试验:
              • ①进行两次蓄水试验:第一次，防水层施工完毕:第二次，设备与饰面层施工完毕。
              • ②蓄水时间为24h。
              • ③蓄水高度:最浅处不应低于10nml。
            • 2. 墙面间歇淋水试验:应达到30min 以上不渗漏。
        • + - 三、屋面防水
          • 屋面淋水或蓄水试验:
            • 1.一般的( 不能做蓄水检验的屋面)，屋面防水层完工后，应在雨后或持续淋水2h 。
            • 2. 有可能做蓄水检验的屋面，其蓄水时间不应少于24h 。
            • 达到无渗漏、积水和排水系统畅通。
      • + - 四、装饰装修工程
        • 一、抹灰工程
          • + - 抹灰前的准备工作:
            • 1.混凝土面层的处理:
              • ①表面应凿毛或洒水湿润后涂刷1 : 1 水泥砂浆(加适量胶粘剂) 。
              • ②加气混凝土应在湿润后，边刷界面剂便抹强度不小子师的水泥混合砂浆。
              • ③局部凹凸不平，事先应剔平或用1 :3 水泥砂浆补平。
            • 2. 浇水湿润注意要点:抹灰前一天， JI院墙自上而下、分次浇水，达到既湿润又不泌水为宣。
            • 3 . 基层质量验收合格后方可抹灰。
            • 4 . 阳角做法
              • 墙面、柱面、门洞口阳角的做法应符合设计要求:
              • 设计无要求时，应采用1: 2 水泥砂浆做暗护角，其高 度不应低于2日1，每侧宽度不应小子50mm。
          • + - 抹灰注意要点:
            • 1.分层抹灰
              • ①水泥砂浆和混合砂浆抹灰时，应待前一层凝结后方可抹后一层:
              • ②石灰砂浆，要在前一层七八成干后方可抹后一层。
              • ③底层的抹灰层强度不得低于面层抹灰强度。
            • 2. 抹灰的养护时机:抹灰24h 后进行湿润养护。
        • + - 二、轻质隔墙工程
          • 骨架隔墙的龙骨安装要求:
            • 1.竖龙骨的安装要点:
              • 有门窗洞口时，从门窗洞口开始分别向两侧排列安装。
              • 无洞口，从一侧向另一侧顺序安装。
            • 2. 安装通贯横撑龙骨的要求: 低于3m 安装一道， 3"'5m 高度的隔墙安装2'''3 道。
          • + - 罩面板的安装顺序:
            • 1.先安装一侧罩面板:
              • ①直竖向铺设，长边接缝应落在竖向龙骨上;曲面罩面板可横向铺设。
              • ②罩面板就位后，用自攻螺钉将板材与轻钢龙骨固定。
            • 2 . 安装填充材料。
            • 3. 安装另一侧罩面板:
              • ①装配板缝与对面的板缝不得布在同一根龙骨上。
              • ②双层石膏板，第2 层的板缝与第1 层板的板缝应错开，接缝不得布在同一根龙骨上。内、外层板应采 用不同的钉距，错开铺钉。
          • 板材隔墙施工要点:
            • + - 1.安装隔墙板的方向:
              • ①有门洞口时，从门洞口处向两端依次进行，门洞两侧直用整块版。
              • ②无洞口的墙体，应从一端向另一端顺序安装。
            • 2. 加气混凝土隔板的接缝处理:
              • ①板缝在填缝前应用毛刷蘸水湿润，填缝时应由两人在板的两侧同时把缝填实。
              • ②填缝材料:用石膏或膨胀水泥。
              • ③隔墙板安装10d 后，检查缝隙是否粘结良好。
        • 三、吊顶工程
          • + - 吊杆的安装要点:
            • 1.固定方式:膨胀螺栓。
            • 2. 吊杆距主龙骨端部距离不得大于300mm; 否则增加吊杆。
            • 3. 吊杆直径要求:根据①是否上人、②是否大子1m，选用#6#8#10 吊杆。
            • 4. 增设附加吊杆的情形。
              • ①一般灯具、风口、检修口( 安装到吊顶工程龙骨上)，应设附加吊杆(与龙骨连接起加固作用) 。
              • ②重型灯具、吊扇严禁直接安装在吊顶工程的龙骨上，必须增设附加吊杆( 直接和设备连接) 。
            • 5. 吊杆长度大于1.5m，应设置反向支撑。
          • + - 龙骨安装要点:
            • 1.主龙骨宣平行房间长向安装。
            • 2. 主龙骨应沿着房间的短向跨度起拱。
            • 3. 主、次龙骨的接长不得搭接，应采取对接。
          • + - 石膏板的安装要点:
            • 1.纸面石膏板的长边应沿纵向次龙骨铺设。
            • 2. 应采用自攻螺钉从一块板的中间向四边进行固定，不得多点同时作业。
            • 3. 螺钉钉头直略埋入板面，但不得损坏纸面，钉眼应作防锈处理并用石膏腻子抹平。
            • 4. 安装双层石膏板时，面层板与基层板的接缝应错开，不得在一根龙骨上。
        • + - 四、地面工程
          • 铺设要点:
            • 1.刷胶:在铺设塑料板的基层位置和塑料板的背面各涂刷一道胶;涂刷基层时，应超出分格线10mm， 涂刷厚度应小子1mm
            • 2. 铺塑料板:待胶干燥至不沾手为直开始铺设，且按弹线位置沿轴线由中央向四周铺贴。
            • 3. 板缝焊接:直在铺贴完48h 后施焊。
        • 五、建筑幕墙工程
          • 预埋件的安装部位:
            • 轻质填充墙、砌体结构不应作幕墙的支撑结构，
            • 应在连接部位的主体结构上增设钢筋混凝土或钢结构梁、 柱。
          • + - 后置埋件的安装:
            • 1.锚固位置:
              • ①锚栓不得(仅)布置在混凝土保护层中;
              • ②锚栓不直设置在钢筋密集区域( 如承重梁的底部、墙柱交叉部位等)，应避开受力主筋，钻孔不得伤 及钢筋。
            • 2. 锚固深度:锚固深度不得包括混凝土的饰面层或抹灰层。
            • 3. 废孔处理:对于废孔，应用化学锚圄胶或高强度树脂水泥砂浆填实。
            • 4 . 后置埋件的试验:
              • ( 1) 应进行承载力现场试验，其值应大于设计值的一倍。
              • ( 2) 必要时做极限拉拔试验。
          • 幕墙安装前对主体结构的测量:
            • 1.安装前对主体结构的复测内容:轴线位置、各层标高、垂直度、混凝土结构构件局部偏差和凹凸程度。
            • 2. 主体结构存在微小偏差的处理:可采取调整幕墙的分格和平面位置分段进行消化，并将施工图调整意见提交建设、设计等有关单位，经 修改施工图后方可施工。
            • 3 . 预埋件位置偏差过大或未设预埋件时:应制定补救措施或可靠连接方案，并将施工图调整意见提交建设、设计等有关单位，经修改施工图后方 可施工。
            • 4. 由于主体结构施工偏差而妨碍幕墙施工时:应会同建设、土建单位采取相应措施。
    • 事后
      • + - 一、隐蔽工程验收
        • 1.监理人未按时进行检查，也未提出延期要求的，视为隐蔽工程检查合格，承包人可自行完成覆盖工作。
        • 2. 隐蔽工程隐蔽后，监理工程师对质量有疑问的，可要求承包人对己覆盖的部位进行钻孔探测或揭开重 新检验， 承包人应遵照执行，并在检验后重新覆盖恢复原状。
        • + - 3. 重新检查验收造成费用增加和工期延误的处理:
          • ( 1) 遵守验收程序( 即1 的要求)的:
            • ①符合合同要求的，由建设单位承担由此增加的费用和工期延误，并支付承包人合理利润。
            • ②不符合合同要求，由此增加的费用和工期延误由承包人承担。
          • ( 2) 承包人私自覆盖的:不论合格与否，由此增加的费用和工期延误均由承包人承担。
      • + - 二、地基基础分部工程的验收.
        • 验收程序:按施工企业自评、设计认可、监理核定、业主验收、政府监督的程序进行。
        • + - 验收的组织
          • 1.施工单位自检，并提供自评报告:由项目经理和施工单位负责人审核、签字、盖章。【同主体】
          • 2. 监理单位进行质量评价，提供质量评估报告: 【同主体】 由总监理工程师和监理单位有关负责人审核、签字、盖章。
          • + - 3 . 正式验收:
            • ①组织者:由总监理工程或建设单位项目负责人(也是基础验收小组的组长，并由其主持验收会议) 。 【同主体】
            • ②参加单位:建设、勘察(勘察单位项目负责人主体工程验收时可以不参加)、设计、监理、施工单位 参加。
            • ③验收前3 个工作日通知质监站。【同主体】
          • 4. 验收后，参建责任方签署基础验收质量记录，应在签字盖章后3 个工作日内由项目监理人员报送质监 站存档。【同主体】
      • + - 三、主体结构分部工程验收.
        • 1.主体结构实体检验项目:混凝土强度、钢筋保护层厚度、结构位置与尺寸偏差、合同约定的项目。
        • 2. 结构实体检验的实施。
          • ( 1) 如何实施:由监理单位组织施工单位实施，并见证实施过程。
          • ( 2) 检验单位:
            • ①结构位置与尺寸偏差，可由施工单位检验;
            • ②其余项目，应由具有相应资质的检测机构完成。
          • ( 3) 检验方法:
            • ①非破损法:超声波探伤、X 射线探伤、γ 射线探伤。
            • ②破损法:回弹一取芯法:
          • 结构实体混凝土强度直采用同条件养护试件进行检验，当未取得同条件养护试件或同条件养护试件强度 不符合要求时，可采用回弹一取芯法进行检验。
      • + - 四、室内装饰装修工程(室内环境质量)的验收
        • 室内环境质量验收时间:应在完工至少7d 后，工程交付使用前进行。
        • 室内污染物种类:氧、氨、苯、甲醋、TVOC 。
        • + - 抽检数量:
          • ( 1) 一般情况: 不少子房间数的5%，并不少于3 间:房间总数少于3 间，应全数检测。~p max {59'0, 3 间}
          • ( 2) 进行样板间检测且合格的: 抽检数量减半(5%/ 2) ，且不少子3 间。~p max { 2. 邸， 3 间}
          • ( 3) 检测不合格时: 再次检测时，最小抽检量=第一次抽检数量X2+ 同类型房间+原不合格房间。
        • 检测点的位置: 距内墙面不小子0.5m，距楼地面高度0.8~1.5m，均匀布置，且避开通风道与通风口。
        • 检测条件:
          • ①采用集中空调的民用建筑工程: 在空调正常运转的条件下进行。
          • ②采用自然通风的民用建筑工程: 检测氛浓度时，应在对外门窗关闭24h 后进行: 检测其他污染物浓度时，应在对外门窗关闭1h 后进行。
      • + - 五、竣工验收
        • + - 竣工验收的组织。
          • 1.竣工预验收:由5总监理工程师组织。
          • 2 . 正式验收:
            • ①施工单位向建设单位提交工程竣工验收报告，申请工程竣工验收，工程竣工报告须经总监理工程师签 署意见。
            • ②建设单位应当在工程竣工验收7 个工作日前通知质监站。
            • ③建设单位组织工程竣工验收。
        • 工程资料组卷和移交。
          • 1.组卷要求:
            • ①专业承包单位施工资料由专业承包单位负责，并单独组卷。
            • ② 电梯应按不同型号每台电梯单独组卷。
          • 2 . 移交顺序:
            • ①施工单位、监理单位分别向建设单位移交资料;
            • ②实行施工总承包的，各专业承包单位应向施工总承包单位移交施工资料，施工总承包单位向建设单位 移交。
            • ③建设单位向城建档案馆移交工程资料。
          • 3. 工程资料应该为原件，当为复印件时如何处理:提供单位应在复印件上加盖单位印章，并应有经办人签字及日期，提供单位应对资料的真实性负责。
